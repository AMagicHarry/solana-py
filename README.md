# PySerum
Python client library for interacting with the Project Serum DEX.

# Development
## Setup
1. Install pipenv.
```sh
brew install pipenv
```
2. Install dependencies.
```sh
pipenv install
```
3. Activate the pipenv shell.
```sh
pipenv shell
```

## Lint
```sh
make lint
```

## Test
```sh
make test
```

## Using Jupyter Notebook
```sh
make notebook
```
